# Leads-Tracker
A simple and usefull chrome Extension that saves the current tab URL and input data.
## What it actually do
- This is basically a chrome extension like other extensions.
- It is used to do very crucial and usefull thing like saving leads.
- firstly we have 3 buttons in this and 1 input field.
- The first button saves whatever we type or paste in input field.
- The second button saves the current tab of the active Window.
- The third button obviously for the delete option.
## How it works 
- Basically it uses Tech stacks as follows:
    - HTML
    - CSS
    - Jvascript
    - JSON
- It uses LocalStorage for rendering the data.
- It uses chrome api for getting the current window information.

## How to set this up
- Firstly download the zip file of this repo.
- then go to url:chrome://extensions/
- ON developer mood.
- You can see options like load unpacked.
- There you select the extracted file.
- then enjoy using application.(You will find it in the extensions wallet)
